Example 1: <br> <br> 
Input: s = "bcabc" <br> 
Output: "abc" <br> 
Example 2: <br> <br> 
Input: s = "cbacdcbc" <br> 
Output: "acdb" <br> <br> <br> 
Constraints: <br> <br> 
1 <= s.length <= 104 <br> 
s consists of lowercase English letters. <br>
