## Given two integers n and k, return all possible combinations of k numbers out of the range [1, n]. <br> <br> 
You may return the answer in any order. <br> <br> <br> <br> 
Example 1: <br> <br> 
Input: n = 4, k = 2 <br> 
Output: <br> 
[ <br> 
  [2,4], <br> 
  [3,4], <br> 
  [2,3], <br> 
  [1,2], <br> 
  [1,3], <br> 
  [1,4], <br> 
] <br> 
Example 2: <br> <br> 
Input: n = 1, k = 1 <br> 
Output: [[1]] <br> <br> <br> 
Constraints: <br> <br> 
1 <= n <= 20 <br> 
1 <= k <= n <br>
